# Contributing

Thank you for your interest in contributing to the ag.sh repository. Please read the [code of conduct](./CODE_OF_CONDUCT.md).

## Making Changes

Please provide changes through a pull request with a detailed titled and description that explains the changes and provides context.

Pull requests will require review and approval.

## Reporting a Problem

If you're having a problem, please check the Issues tab for any issues that others may have seen before. If you can't find a fix, create a new issue with as much information as possible to help understand the problem. Sample projects are welcome to help investigate any problems quicker.
